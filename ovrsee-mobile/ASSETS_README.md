# Assets Required

The app references several image assets in `app.json`. You'll need to create these:

## Required Assets

1. **assets/icon.png** (1024x1024)
   - App icon for iOS and Android
   - OVRSEE branded icon

2. **assets/splash.png** (2048x2048)
   - Splash screen image
   - Dark background (#0F172A) with OVRSEE logo

3. **assets/adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground

4. **assets/favicon.png** (48x48)
   - Web favicon

## Generating Assets

You can use tools like:
- [Expo Asset Generator](https://docs.expo.dev/guides/app-icons/)
- [Icon Generator](https://www.appicon.co/)
- Or create manually using the OVRSEE logo

## Temporary Solution

For development, you can use placeholder images or the Expo default icons. The app will work with placeholder assets during development.
